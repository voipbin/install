# DNS & Domain Configuration Guide Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add `voipbin-install dns` command that prints a general-purpose DNS and domain configuration guide, and surface the DNS records section in the `apply` completion output.

**Architecture:** New `scripts/commands/dns.py` holds `cmd_dns()` and a shared `print_dns_guide()` helper that renders three sections (DNS records, K8s ConfigMap, Kamailio VM env). `apply.py` calls the same helper after its success result box. All output uses the existing Rich display helpers (`console`, `print_header`, `print_step`, etc.) from `scripts/display.py`. No live queries, no mutations.

**Tech Stack:** Python 3.11+, Rich (already a dependency), Click (already used in `scripts/cli.py`), pytest.

```python
# scripts/commands/dns.py
"""The 'dns' command: DNS and domain configuration guide."""

from scripts.display import console, print_header


_SUBDOMAINS = ["api", "admin", "talk", "meet", "sip"]
_EXAMPLE_DOMAIN = "example.com"
_EXAMPLE_IP = "1.2.3.4"


def print_dns_section1() -> None:
    """Section 1: DNS A records guide."""
    print_header("DNS Records")
    console.print("  [dim]" + "─" * 50 + "[/dim]")
    console.print()
    console.print("  VoIPBin requires the following DNS A records at your registrar.")
    console.print("  All subdomains point to the same load balancer IP.")
    console.print()
    for sub in _SUBDOMAINS:
        fqdn = f"{sub}.{_EXAMPLE_DOMAIN}"
        console.print(f"    [bold]{fqdn:<28}[/bold] A    {_EXAMPLE_IP}")
    console.print()
    console.print("  For [bold]auto[/bold] DNS mode: delegate your domain to the GCP nameservers")
    console.print("  printed after apply completes. GCP then manages the A records.")
    console.print()
    console.print("  DNS propagation can take up to 48 hours.")
    console.print("  Once complete, run: [bold]voipbin-install verify[/bold]")
    console.print()
```
